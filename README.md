# Front-Dockzilla

Фронтенд проект с формой логина на главной странице.

## Запуск проекта фронтенд
Эта часть говорит о том, как запустить frontend сервиса.
```bash

# Установка зависимостей
npm install

# Запуск в режиме разработки (сервер запустится на http://localhost:3000)
npm run dev

# Или запуск с автоматическим открытием браузера
npm start
```
Запустится http://127.0.0.1:3000/ , там можно авторизоваться и работать с файлами.(при работающем бэкенде)
http://127.0.0.1:3000/weather.html это фронт для работы с эндпоинтом погоды

Так же если перейти в sorting_liquids/main_sort.js можно найти решение сортировки. 
Его можно запустить любым встроеным обработчиком, или воспользовавшись встроенными возможностями 
этого сервиса, например:  node src/sorting_liquids/main_sort.js
