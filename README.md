# Front-Dockzilla

Фронтенд проект с формой логина на главной странице.

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки (сервер запустится на http://localhost:3000)
npm run dev

# Или запуск с автоматическим открытием браузера
npm start
```

## Структура проекта

```
front-dockzilla/
├── src/
│   ├── index.html        # Главная страница с формой логина
│   ├── styles.css        # Стили формы логина
│   └── app.js           # JavaScript логика обработки формы
├── public/              # Папка для статических файлов
├── package.json         # Конфигурация проекта и зависимости
└── README.md           # Документация
```
